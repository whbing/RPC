# 使用方法
分别启动Simple-S，Simple-C main方法，即可检测运行结果：
`Simlpe-S`:
```bash
服务开启
/127.0.0.1-connected
/127.0.0.1-connected
```
`Simple-C`:
```bash
say:hello,whb
Person:Person [name=whb, age=18]
```
